# Stateless Pager component.
## Usage
```
<Pager current={3}
       total={20}
       visiblePages={5}
       onPageChanged={this.handlePageChanged} />
```

## How it looks like*
```
First | Prev | ... | 6 | 7 | 8 | 9 | ... | Next | Last
```

* Bootstrap 3.0 is required by default, but you can replace it with your own css.
